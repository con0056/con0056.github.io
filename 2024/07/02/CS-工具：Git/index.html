1-6 初次运行Git前的配置

Git - 初次运行 Git 前的配置 (git-scm.com)

  • 配置变量存储在三个层次：每个级别会覆盖上一层级别的配置

    • 系统上每个用户
    • 当前用户：使用 --global 选项
    • 当前仓库

  • 设置用户信息

    • 当你想针对特定项目使用不同的用户名称与邮件地址时，可以在那个项目目录下运行没有 --global 选项的命令来配置

  • 命令：git config｜注意这里的 config 是一个 <verb>

    • 查看配置信息
      • 所有：--list
      • 某一配置：<key>
      • 可选：--show-origin，展示来源文件
    • 在不同级别设置配置变量
      • --system
      • --global
      • --local
