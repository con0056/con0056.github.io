四、对象的行为：方法使用实例变量

用时：1h

猜测讲到封装了。开头引言表达的核心思想是：状态和行为相互影响（也就是实例对象和方法）。

  • 方法（参数传递的方法：按值传递）

    • 实参
    • 形参：获得实参的一个副本

  • 返回值

    • 需与声明的返回类型兼容

  • 隐式宽化：小东西放到大杯子

  • 隐式转换包括宽化

  • 封装：一个类的方法必须有：获取方法、设置方法、方法

    • 实例变量 private
    • 方法 public
      • Getter
      • Setter

  • 实例变量和局部变量

    • 实例变量
      • 类中声明，每个类型有默认值，无需初始化
      • 默认值
        • 基本类型：数值 & char 是 0；boolean 是 false
        • 引用类型：null
    • 局部变量
      • 方法中声明，必须初始化
      • 形参是特殊的实例变量

  • 比较变量

    • 相等：.equals方法

      • 对象之间的相等取决于对这个对象类型，最重要的是什么

    • 相同：==

    • python中

      • 相等：==
      • 相同：is operator
